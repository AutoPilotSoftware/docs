## Можливості платформи

### Безпека
- 🔁 **Автоповернення** при невідповідності країни
- 🔄 **Перезавантаження** при збої проксі (reKYC)
- 🔀 **Ротація портів** при блокуванні проксі
- ⏰ **Дедлайни** — автоперепризначення завдання при простроченні

### Додатково
- 💳 Платіжна система з балансом у **USDT**
- ⭐ Рейтинги селлерів з рівнями (Gold / Silver / Bronze)
- 📊 Аналітика: витрати, графіки активності, % успіху
- 🔍 Фільтри за країнами, біржами, масові замовлення
- 🌍 Локалізація — EN / RU / UA
- 💬 Вбудований анонімний чат із селлерами
