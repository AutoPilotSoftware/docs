# Подписка AutoPilot KYC

Бот для автоматизации KYC на **Bybit** и **MEXC** — загружаете аккаунты → автоматическая валидация → распределение селлерам → верификация.

```mermaid
flowchart LR
    A["📤 Загрузка аккаунтов"] --> B["🔍 Автовалидация"]
    B --> C["👥 Распределение селлерам"]
    C --> D["✅ Верификация"]
```

## Две модели работы

| | Глобальная сеть | Своя команда |

| Селлеры | 40+ стран, авто-распределение | Свои воркеры, своя цена |
| Ценообразование | Конкуренция между локальными селлерами | Вы устанавливаете цену |
| Выплаты | Автоматически on-chain | Автоматически on-chain |
| Контроль | Рейтинги, дедлайны, авто-замена | Полный контроль команды |

## Возможности платформы

### Безопасность
- 🔁 **Автовозврат** при несовпадении страны
- 🔄 **ReKYC** — повторная верификация лица тем же селлером по запросу биржи
- 🔀 **Ротация портов** при блокировке прокси
- ⏰ **Дедлайны** — автопереназначение задачи при просрочке

### Дополнительно
- 💳 Платёжная система с балансом в **USDT**
- ⭐ Рейтинги селлеров с уровнями (Gold / Silver / Bronze)
- 📊 Аналитика: расходы, графики активности, % успеха
- 🔍 Фильтры по странам, биржам, массовые заказы
- 🌍 Локализация — EN / RU / UA
- 💬 Встроенный чат с селлерами (анонимный)
